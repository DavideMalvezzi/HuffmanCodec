FileInfo* createNewFileInfo ( const gchar *  path)

Crea una nuova istanza di FileInfo a partire dal path di un file.

FileInfo::name viene inizializzata a con il nome del file estratto dal path in input
FileInfo::size viene inizializzata con la dimensione corrente del file
FileInfo::comprSize viene inizializzata a 0
FileInfo::keywords viene inizializzata a NULL
